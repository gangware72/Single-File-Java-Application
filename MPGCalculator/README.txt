This was assignment was completed Spring 2017 as a warm up to Data Structures.

Requirements:
Calculate the average MPG given user input for (1) miles traveled and (2) Gallons consumed.

Compile and run CalculateMPG with java.

Find folder containing MPRGCalculator and type 'java CalculateMPG'

To compile:
(1) Type 'javac CalculateMPG.java'
(2) Type 'java CalculateMPG'
